# Excalidraw 使用指南

## 一、简介

Excalidraw 是 Obsidian 中的手绘风格白板插件，可以绘制流程图、思维导图、架构图、示意图等，风格独特且美观。

---

## 二、安装与基本设置

### 安装
> 设置 → 社区插件 → 浏览 → 搜索 **Excalidraw** → 安装 → 启用

### 核心设置

> 设置 → Community plugins → Excalidraw

| 设置项 | 推荐值 | 说明 |
|--------|--------|------|
| **Auto-save** | ✅ | 自动保存 |
| **Default save location** | `04-Excalidraw` | 新建白板存放位置 |
| **Embed font** | ✅ | 嵌入字体确保跨设备显示 |
| **Theme** | 跟随系统 | 深色/浅色主题 |

---

## 三、创建白板

### 方式 1：命令面板
```
Ctrl/Cmd + P → Excalidraw: Create new drawing
```

### 方式 2：文件图标
左侧边栏点击 Excalidraw 图标 → 新建白板

### 方式 3：在笔记中嵌入
在任意笔记中输入：
```markdown
![[我的白板.excalidraw]]
```

| **图片** | - | 插入图片 |

## 五、常用操作

### 1. 画流程图

```
步骤：
1. 按 R 画矩形 → 输入文字
2. 按 A 画箭头连接
3. 按 R 画菱形（判断）→ 输入"是/否"
4. 继续用箭头连接各节点
```

### 2. 编辑文字
- **双击**已有元素 → 进入文字编辑模式
- 或选中元素后按 `Enter`

### 3. 连接线吸附
- 画箭头时靠近元素会自动**吸附到边框**
- 移动元素时连接线**自动跟随**

### 4. 组合元素
- 选中多个元素（`Shift + 点击` 或拖拽框选）
- 按 `Ctrl/Cmd + G` 组合
- 组合后可以整体移动

### 5. 对齐与分布
选中多个元素后：
```
顶部工具栏 → 对齐选项
├── 左对齐 / 水平居中 / 右对齐
├── 顶部对齐 / 垂直居中 / 底部对齐
└── 水平分布 / 垂直分布
```

## 六、样式自定义

### 修改元素样式
选中元素 → 右侧面板或顶部工具栏：

| 属性 | 说明 |
|------|------|
| **Stroke color** | 边框颜色 |
| **Background color** | 填充颜色 |
| **Stroke width** | 线条粗细 |
| **Stroke style** | 实线/虚线/点线 |
| **Roughness** | 手绘感强度（0=平滑，2=粗糙） |
| **Opacity** | 透明度 |
| **Font size** | 字体大小 |

### 预设颜色
Excalidraw 提供了丰富的手绘风格配色，可以直接点击使用。

## 七、在笔记中嵌入白板

### 完整嵌入
```markdown
![[我的白板.excalidraw]]
```
在笔记中直接显示完整白板。

### 区域嵌入（只显示部分内容）
```markdown
![[我的白板.excalidraw#^element-id]]
```

### 获取元素 ID
1. 在 Excalidraw 中选中元素
2. 右侧面板会显示该元素的 ID
3. 复制 ID 用于嵌入

## 八、实用技巧

### 快捷键速查

| 操作 | 快捷键 |
|------|--------|
| 撤销 | `Ctrl/Cmd + Z` |
| 重做 | `Ctrl/Cmd + Shift + Z` |
| 复制 | `Ctrl/Cmd + C` |
| 粘贴 | `Ctrl/Cmd + V` |
| 删除 | `Delete` / `Backspace` |
| 全选 | `Ctrl/Cmd + A` |
| 缩放 | `Ctrl/Cmd + 滚轮` |
| 平移画布 | 按住 `Space + 拖拽` |
| 锁定工具 | 双击工具图标 |

### 手绘风格调整
- **Roughness = 0**：平滑线条（适合正式文档）
- **Roughness = 1**：轻微手绘感
- **Roughness = 2**：强烈手绘感（草图风格）

### 快速复制元素
- 选中元素 → `Alt + 拖拽` → 快速复制

### 等比例绘制
- 按住 `Shift` 画矩形 → 正方形
- 按住 `Shift` 画圆 → 正圆

## 九、导出与分享

### 导出图片
1. 点击左上角菜单（三横线）
2. 选择 **Export image**
3. 选择格式：PNG / SVG
4. 选择是否包含背景

### 导出为文件
- **.excalidraw** 文件可以分享给他人
- 对方安装 Excalidraw 即可打开编辑

## 💡 Tips

1. **自动吸附**：画连接线时靠近元素会自动吸附，移动时连接线跟随
2. **嵌入笔记**：用 `![[文件名.excalidraw]]` 在笔记中直接显示白板
3. **协作**：.excalidraw 文件是纯 JSON，可以版本控制
4. **主题跟随**：Excalidraw 可以跟随 Obsidian 的深色/浅色主题
