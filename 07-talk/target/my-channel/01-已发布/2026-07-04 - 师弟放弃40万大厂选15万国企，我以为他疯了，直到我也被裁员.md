## 📖 文案内容

```
师弟放弃40万大厂，选了15万的国企。

我当时觉得他疯了。

直到前年，我也被裁了。

北邮硕士，阿里P7，5年大厂经验，在济南找工作不是随便挑？

投了30份简历，只有3个面试，甚至还有问我做不做外包。

那个时候我才真正明白——应届生身份，是我这辈子丢过最贵的东西。

垄断型国企、地方人才引进，只对应届生开放，而且门槛越来越高。

你一旦丢了这张门票，再想进去，撞破头连个门都没有。

19年，刚工作，P8主管，31岁，入手1000万的房子，说要在阿里干到退休。那时候80%的同学都选互联网，去国企那是能力不行的才去的。

现在呢？

上个月飞猪裁了40%。

AI时代，今天入职，没准隔俩月就得离职。

毕业前5年，我在阿里赚了快300万。

但是现在，去大厂混几年捞一笔钱的时代，已经一去不复返了。

2026年，互联网大厂还能去吗？

如果你和我一样，是普通家庭的孩子，也没有在学校就光芒四射——

那选大厂之前，一定好好想想，你手里的应届生身份。

普通家庭的孩子真的没有赌的资本。

其实不去大厂，踏踏实实，在二线城市找个稳定的工作，夫妻俩人公积金覆盖房贷，日子过得也很舒服。

这是阿里到国企第12篇，下一期，什么样的人，我劝他应该去大厂

👇 评论区聊聊：你当年选大厂还是国企，现在后悔了吗？
```

## 💡 创作思路

### 视角选择
**"应届生身份是最被低估的资产"**

同行讲烂了"大厂vs国企哪个好"，换个视角：
- 不讲"选哪个"，讲**"选之前先想清楚你丢的是什么"**
- 有立场、有紧迫感、有具体后果

### 结构拆解
```
钩子1（0-3秒）：师弟40万vs18万 → 数字对比，抓住注意力
钩子2（4-8秒）：我当时觉得他疯了 → 制造悬念
钩子3（9-15秒）：北邮硕士+阿里P7 → 建立可信度
钩子4（16-20秒）：投30份只有3个面试 → 冲突转折，信息缺口
钩子5（21-25秒）：应届生身份，是我这辈子丢过最贵的东西 → 核心观点，情绪爆发
```

### 素材串联逻辑
```
师弟的选择（钩子）
    ↓
我的反转（被裁员+找不到工作）
    ↓
核心观点（应届生身份最贵）
    ↓
为什么（国企只对应届生开放）
    ↓
时代背景（P8主管+飞猪裁员）
    ↓
明确建议（先想清楚再用应届生身份）
```
