## 六、经历素材×热点 组合公式

后续挖掘新选题时，用这个公式：

```
新选题 = 你的经历素材（exact数字/场景） × 当前社会热点情绪
```

**步骤**：
1. 从经历素材库找一条exact经历
2. 从当前热点话题找一个匹配的社会情绪
3. 用3维漏斗打分
4. ≥80分就做

**例子**：
- 经历：「报销系统难用，俩月审批还没走完」
- 热点：「国企效率低」（持续热点）
- 组合：《在阿里报销一键到账，在国企俩月还没审批完》
- 3维打分：方向锚定✅ + 搜索热词✅ + 数字锚点✅ + 情绪锚点✅ + 反差✅ + 独特性✅ = **100分**

## 七、经历素材库的后续使用建议

### 7.1 每次选题前先查经历素材库

按经历类型分类查找：
- 需要数字锚点 → 查「阿里薪资」「国企薪资」「裁员经历」
- 需要场景冲突 → 查「工作节奏」「对比经历」
- 需要心理模式 → 查「家庭背景」

### 7.2 每次发布后补充新经历

每篇口播稿发布后，把里面用到的经历素材在经历素材库里标注「已用于《XXX》」，避免重复使用同一经历。

### 7.3 经历素材持续挖掘

每周花10分钟回顾本周经历，看有没有新的exact经历可以补充到素材库里。素材库越厚，选题越不愁。
