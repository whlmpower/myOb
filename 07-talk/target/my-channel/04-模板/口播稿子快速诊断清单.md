# 口播稿子快速诊断清单

> **使用场景**：发布前快速自检，5分钟完成诊断
> **最后更新**：2026-07-04

---

## 快速诊断流程

```
[ ] 话题设置 → [ ] 标题设置 → [ ] 文案钩子 → [ ] 评论区引导
```

---

## 一、话题设置（3分钟）

### 必问3个问题

- [ ] **这个选题的核心冲突是什么？**（用一句话说清楚）
- [ ] **这个选题的目标人群是谁？**（越具体越好）
- [ ] **这个选题的讨论点在哪里？**（用户会讨论什么）

### 自检标准

| 检查项 | 通过标准 |
|--------|----------|
| 定位清晰 | 一句话能说清内容核心 |
| 人群明确 | 能说出"这个内容是为XX人写的" |
| 有讨论点 | 看完有想评论的冲动 |
| 符合系列定位 | 与系列主题高度相关，不漂移 |

### 常见问题

- 🔴 **话题太泛**："择校真相" → 改成"北邮在山东不如青大"
- 🔴 **定位漂移**：中途插入无关内容 → 单期只打一个点
- 🔴 **人群模糊**："所有人" → 改成"山东高考考生"

---

## 二、标题设置（1分钟）

### 必问3个问题

- [ ] **标题里有数字吗？**（数字增加可信度）
- [ ] **标题里有情绪爆点吗？**（后悔/崩溃/震惊/逆袭）
- [ ] **标题里有目标人群吗？**（山东考生/大厂人/国企人）

### 标题公式（三选一）

| 公式 | 模板 | 示例 |
|------|------|------|
| **反差公式** | [A] + [B] + 我才懂[X] | 放弃3所985选北邮，在阿里5年才知道... |
| **人群公式** | [人群]必看：A比B更管用 | 山东考生别踩坑：北邮在山东... |
| **悬念公式** | [反常识行为]，[结果] | 我放弃山大选北邮，回山东肠子都悔青了 |

### 自检标准

| 检查项 | 通过标准 |
|--------|----------|
| 有数字 | 包含具体数字（3所/5年/100万） |
| 有情绪 | 包含情绪词（后悔/崩溃/震惊/逆袭） |
| 有人群 | 明确指向目标人群 |
| 有反差 | 存在明显对比或冲突 |

---

## 三、文案钩子（1分钟）

### 必问4个问题

- [ ] **前3秒有画面感吗？**（具体事件/场景，不是抽象道理）
- [ ] **前5秒有冲突吗？**（反差/矛盾/悬念）
- [ ] **中间有钩子吗？**（每30秒一个"小钩子"）
- [ ] **结尾有行动引导吗？**（关注/评论/收藏）

### 钩子密度检查

| 位置 | 必备钩子类型 | 示例 |
|------|------------|------|
| **0-5秒** | 情绪爆点 | HR问我北邮是985还是211，我瞬间懵了 |
| **5-30秒** | 冲突钩子 | 我放弃了3所985选了北邮，现在肠子都悔青了 |
| **30秒-2分钟** | 案例钩子 | 举个例子，26年青岛青选计划... |
| **2-3分钟** | 情绪爆点 | 我当年要是知道这个，打死我也不会选北邮 |
| **3-4分钟** | 行动钩子 | 如果你也在山东找工作，这个真相你必须知道 |

### 自检标准

| 检查项 | 通过标准 |
|--------|----------|
| 5秒钩子 | 前5秒有具体画面/冲突/悬念 |
| 钩子密度 | 每30-60秒有一个小钩子 |
| 情绪弧线 | 有"发现问题→感受矛盾→反思"的过程 |
| 结尾引导 | 有明确的关注/评论/收藏引导 |

---

## 四、评论区引导（30秒）

### 必问3个问题

- [ ] **文案里有评论引导吗？**（"打在公屏上""评论区聊聊"）
- [ ] **有置顶评论准备吗？**（发布后立即置顶）
- [ ] **有自问自答准备吗？**（预设用户问题并回答）

### 评论区运营SOP

```
发布前准备：
  📌 置顶评论（1条）
  💬 自问自答评论（2-3条）
  🔥 神评预设（1-2条）

发布后1小时内：
  1. 立即置顶评论
  2. 用小号发神评
  3. 前10条评论全部回复
```

### 自检标准

| 检查项 | 通过标准 |
|--------|----------|
| 文案内嵌引导 | 有"打在公屏上""评论区聊聊"等语句 |
| 置顶评论准备 | 有1条准备好的置顶评论 |
| 自问自答准备 | 有2-3条预设问答评论 |
