## 🔍 话题预判检查清单

### □ 1. 搜索验证（10 秒法）

**重要修正：** 抖音搜索框下拉列表**不直接展示搜索量数字**。判断热度靠以下 3 个信号：

**信号 1：下拉词排序（最快，10 秒）**

| 排序位置 | 热度判断 | 行动 |
|---------|---------|------|
| **第 1 个** | 高 | ✅ 必做 |
| **第 2-3 个** | 中高 | ✅ 可以做 |
| **第 4-5 个** | 中 | ✅ 可以做 |
| **第 6 个及以后** | 低 | ⚠️ 谨慎 |

**操作：** 输入核心词，看前 5 个下拉词有没有"查询型词"（是什么/多少钱/多少级）。不看数字，只看有没有 + 排第几。

---

**信号 2：巨量算数验证（1 分钟，免费官方工具）**

巨量算数（https://trendinsight.oceanengine.com）能查搜索指数。

**判断标准：**

| 搜索指数 | 热度 | 结论 |
|---------|------|------|
| > 10 万 | 高 | ✅ 优先做 |
| 1-10 万 | 中 | ✅ 可以做 |
| < 1 万 | 低 | ⚠️ 除非有特殊理由，否则不做 |

---

**信号 3：常识判断（零成本，最快）**

| 词 | 常识判断 | 理由 |
|----|---------|------|
| 阿里P7 / P8 / P6 | 高 | 大厂职级是经典搜索词 |
| P7薪资 / 年薪 | 高 | 薪资是永恒热点 |
| 35岁危机 / 裁员 | 高 | 社会情绪+大厂裁员 |
| 年终奖 / 股票 | 高 | 周期性热点 |
| 隐形福利 / 稳定工作 | 低 | 太泛，没有查询型属性 |
| 牢笼 / 快乐吗 | 无 | 抽象概念，搜索量几乎为零 |

### □ 2. 话题类型确认

**T1 高搜索价值型（流量放大器）**

- [ ] 用户主动搜索的高频词
- [ ] 有"是什么""多少钱""多少级"等查询属性
- [ ] 你是有资格回答这个问题的人（亲身经历 / 专业背景）

**T2 高互动讨论型（粉丝筛选器）**

- [ ] 天然有争议性 / 对比冲突
- [ ] 天然引发"我也有类似经历"的共鸣
- [ ] 评论区会自发讨论，不需要你硬引导

**T3 情感共鸣型（调剂用，非主力）**

- [ ] 有情绪共鸣
- [ ] 但搜索价值低
- [ ] 互动价值有限

**T4 纯自嗨型（禁止做）**

- [ ] 只有你能理解，用户不关心
- [ ] 抽象概念，没有具体锚点
- [ ] 搜索下拉词无查询属性

**话题类型结论：** T1 / T2 / T3 / T4
