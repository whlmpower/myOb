# 抖音话题标签选择策略

> 抖音允许加 5 个话题标签。  
> 5 个位置不是平等的——主标签选错了，后面 4 个再好也没用。  
> 话题标签的作用是**告诉算法你的内容属于哪个分类**，不是"让用户搜到你"。

---

## 一、5 个标签的分配逻辑

| 位置          | 权重    | 作用          | 标签类型  | 选择标准         |     |
| ----------- | ----- | ----------- | ----- | ------------ | --- |
| **第 1 个**   | ⭐⭐⭐⭐⭐ | 主标签，决定内容主分类 | 大流量标签 | 搜索量 ≥ 100 万  |     |
| **第 2-3 个** | ⭐⭐⭐⭐  | 精准标签，锁定目标人群 | 精准标签  | 搜索量 10-100 万 |     |
| **第 4-5 个** | ⭐⭐⭐   | 长尾标签，补充流量入口 | 长尾标签  | 搜索量 1-10 万   |     |
|             |       |             |       |              |     |

**第 1 个标签决定算法把你的内容推给谁。选错了，播放量上限锁死。**

---

## 二、你的三层标签池

### 第一层：主标签池（第 1 个位置必选）

**要求：** 搜索量 ≥ 100 万，与核心定位强相关

| 标签 | 适合内容 | 不适用内容 |
|------|---------|-----------|
| `#大厂` | 所有内容 | — |
| `#国企` | 所有内容 | — |
| `#阿里` | 涉及阿里经历的内容 | 与阿里无关的内容 |
| `#职场` | 所有内容 | — |

**你的选择原则：**
- 内容偏阿里经历 → 选 `#大厂` 或 `#阿里`
- 内容偏国企经历 → 选 `#国企`
- 通用内容 → 选 `#大厂`（流量最大）

**固定规则：第 1 个标签永远是你的核心定位标签，不要每篇换。**

---

### 第二层：精准标签池（第 2-3 个位置）

**要求：** 搜索量 10-100 万，精准命中目标人群

| 标签 | 适合内容 | 目标人群 |
|------|---------|---------|
| `#互联网大厂` | 涉及大厂经历/薪资/职级 | 互联网从业者 |
| `#体制内` | 涉及国企/体制内容 | 体制内/想进体制的人 |
| `#大厂日常` | 日常职场观察 | 大厂员工 |
| `#国企日常` | 国企日常观察 | 国企员工 |
| `#职场vlog` | 有具体场景的内容 | 职场年轻人 |
| `#职场干货` | 有具体方法/经验 | 想学习提升的人 |
| `#打工人` | 情感共鸣/自嘲类 | 所有打工人 |
| `#社畜` | 自嘲/共鸣类 | 年轻职场人 |

**你的固定组合（第 1-3 个建议固定不变）：**

```
组合 A（偏大厂内容）：
第 1 个：#大厂
第 2 个：#互联网大厂
第 3 个：#体制内

组合 B（偏国企内容）：
第 1 个：#国企
第 2 个：#体制内
第 3 个：#大厂
```

**第 2-3 个也固定，只换第 4-5 个。**

---

### 第三层：长尾标签池（第 4-5 个位置）

**要求：** 搜索量 1-10 万，高度匹配本期内容的具体角度

这是最灵活的一层，**每篇内容根据具体话题换**。

| 本期内容主题 | 第 4 个标签 | 第 5 个标签 |
|-------------|------------|------------|
| P7 薪资/年薪 | `#阿里p7` | `#薪资` 或 `#年终奖` |
| P8 薪资/晋升 | `#阿里p8` | `#薪资` 或 `#晋升` |
| 月薪对比 | `#薪资` | `#打工人` |
| 福利对比 | `#大厂福利` | `#国企福利待遇` |
| 工作强度/加班 | `#大厂日常` | `#工作强度` |
| 35 岁危机 | `#35岁危机` | `#大厂裁员` |
| 转型/跳槽 | `#跳槽` | `#转型` |
| 会议文化 | `#国企日常` | `#会议` |
| 晋升逻辑 | `#晋升` | `#职场规则` |
| 入职经历 | `#职场vlog` | `#应届生` |
| 年终奖 | `#年终奖` | `#薪资` |
| 股价/股票 | `#阿里股票` | `#职场` |

## 三、你的完整标签方案

### 方案 A：阿里经历类（薪资/职级/内部事件）

```
第 1 个：#大厂
第 2 个：#互联网大厂
第 3 个：#体制内
第 4 个：#阿里p7 或 #薪资 或 #年终奖
第 5 个：#职场干货 或 #职场vlog
```

**示例（P7 薪资篇）：**
```
#大厂 #互联网大厂 #体制内 #阿里p7 #薪资
```
**效果：** 播放量 10.61 万，搜索热词占 71.5%，验证有效。

---

### 方案 B：国企经历类（福利/会议/日常）

```
第 1 个：#国企
第 2 个：#体制内
第 3 个：#大厂
第 4 个：#国企日常 或 #国企福利待遇
第 5 个：#打工人 或 #职场vlog
```

**示例（福利牢笼篇）：**
```
#国企 #体制内 #大厂 #国企福利待遇 #打工人
```
**效果：** 播放量 1.17 万。标签组合本身没问题，但话题（牢笼）搜索价值极低，标签救不了话题。

---

### 方案 C：对比/讨论类（双线并行）

```
第 1 个：#大厂 或 #国企
第 2 个：#互联网大厂
第 3 个：#体制内
第 4 个：#薪资
第 5 个：#打工人 或 #职场vlog
```

**示例（月薪对比篇）：**
```
#大厂 #互联网大厂 #体制内 #薪资 #打工人
```
**效果：** 互动率 2.06%，标签组合有效。

## 四、标签选择禁忌

### ❌ 禁忌 1：每篇换一套全新的标签

**错误：** 这篇 `#大厂 #阿里 #职场 #体制内 #互联网`，下篇换一套别的。

**问题：** 算法无法建立稳定的内容分类，不知道把你的内容推给谁。

**正确：** 第 1-3 个标签固定，只换第 4-5 个。

---

### ❌ 禁忌 2：只用大流量标签，不做精准匹配

**错误：** 所有内容都用 `#大厂 #职场 #互联网 #打工人 #vlog`

**问题：** 标签太泛，算法把你和所有职场博主放在一起，竞争激烈，你跑不过头部账号。

**正确：** 第 1 个用大流量标签，第 2-3 个用精准标签，第 4-5 个用长尾标签。

---

### ❌ 禁忌 3：堆砌无关标签

**错误：** 内容讲国企福利，加 `#互联网大厂 #阿里p7 #年终奖`

**问题：** 算法会认为内容与标签不匹配，降低推荐权重。

**正确：** 第 4-5 个标签必须与本期内容强相关。

---

### ❌ 禁忌 4：用长尾词当主标签

**错误：** 第 1 个标签用 `#国企隐形福利` 或 `#大厂牢笼`

**问题：** 搜索量太低，算法没有足够的用户数据来判断推给谁。

**正确：** 第 1 个必须是搜索量 ≥ 100 万的大标签。

---

### ❌ 禁忌 5：重复使用同一个长尾标签

**错误：** 连续 5 篇都用 `#打工人` 做第 5 个标签。

**问题：** 第 4-5 个标签是给算法补充"本期内容具体角度"的，每篇内容角度不同，标签也应该不同。

**正确：** 第 4-5 个根据本期内容从长尾池里选，尽量不重复。

## 五、标签效果评估

每次发完视频，在创作者中心看这 3 个数据：

| 指标 | 怎么看 | 判断标准 | 不达标怎么办 |
|------|--------|---------|------------|
| **主标签曝光量占比** | 每条标签旁边有曝光数字 | ≥ 30% | 换主标签 |
| **标签点击率** | 曝光量 ÷ 播放量 | > 5% | 换精准标签 |
| **标签涨粉率** | 从该标签进来的用户的涨粉率 | > 平均涨粉率 | 换精准标签 |

**快速判断：**
- 主标签曝光量 < 20% → 算法不认可你这个主标签，换一个
- 标签点击率 < 3% → 用户对这个标签不感兴趣，换一个
- 从某个标签进来的用户脱粉率高 → 这个标签吸引的不是你的目标人群

## 六、你的标签选择速查卡

**每次发布前核对：**

```
□ 第 1 个标签是固定主标签？（大厂 or 国企，不变）
□ 第 2-3 个是固定精准标签？（互联网大厂 / 体制内，不变）
□ 第 4 个标签匹配本期具体内容？
□ 第 5 个标签匹配本期具体内容？
□ 5 个标签都与本期内容强相关？
□ 没有堆砌无关标签？
```

**6 条全过 → 标签合格。**

---

## 七、标签与话题的配合

标签和话题预判是两件事，但需要配合使用：

| 话题类型 | 标签策略 | 示例 |
|---------|---------|------|
| **T1 高搜索** | 主标签 + 精准标签 + 搜索词作长尾 | `#大厂 #互联网大厂 #体制内 #阿里p7 #薪资` |
| **T2 高互动** | 主标签 + 精准标签 + 讨论类长尾 | `#大厂 #互联网大厂 #体制内 #薪资 #打工人` |
| **T3 情感共鸣** | 主标签 + 精准标签 + 场景类长尾 | `#大厂 #互联网大厂 #体制内 #职场vlog #应届生` |
| **T4 禁止** | 不做 | — |

**关键：标签是给算法看的，话题是给用户看的。标签决定推给谁，话题决定用户会不会点进来。**
