# 03｜方法论个人主义：从谁出发看经济

## 这一篇要解决的问题

你提出了华为的案例：产品价值一般，但通过"爱国"营销和饱和式攻击，获得了极高的品牌认可度。这如何解释？主观价值论是不是在说"存在即合理"？

## 正文

### 先回应你的问题

你的案例击中了奥派经济学最核心的方法论：**只有个人能行动，"集体"不能行动。**

华为作为一个公司，本身不能"营销"。是华为里的**个人**在做决策：

- 某个产品经理决定把手机定价 6999 元
- 某个营销总监决定用"爱国"作为传播点
- 某个工程师决定把智驾功能做成某个形态

同样，"群众认可华为"也不是一个集体行为，是**无数个个人**各自做出购买决策。

**方法论个人主义**：所有经济现象，最终都要还原到个人的选择和行动来解释。

### 为什么奥派坚持个人主义

门格尔在《国民经济学原理》中确立了这个原则：**经济学研究的是人的选择，不是物的属性。**

主流经济学（特别是宏观经济学）的问题：

- 说"中国经济增长 5%"
- 说"消费需求不足"
- 说"投资信心低迷"

这些说法把经济当成一个整体机器，好像可以像调节水温一样调节经济。

奥派的反驳：**经济不是机器，是无数个个人互动的结果。**

### 华为案例的个人主义拆解

用方法论个人主义拆解你观察到的现象：

**第一层：华为内部的个人决策**

| 角色   | 个人决策      | 动机            |
| ---- | --------- | ------------- |
| 产品经理 | 定价 6999 元 | 猜测目标用户愿意付这个价格 |
| 营销总监 | 主打"爱国"牌   | 猜测这个叙事能打动目标人群 |
| 研发主管 | 智驾功能取舍    | 在成本、技术、时间之间权衡 |

这些决策都是**个人的主观判断**，不是"华为"这个抽象实体的意志。

**第二层：消费者的个人决策**

为什么有人愿意买 6999 元的华为手机？

- 用户 A："我认同华为代表的中国技术突破，这本身有价值"
- 用户 B："我不关心参数，我用华为有面子"
- 用户 C："我被小米的负面新闻影响了，不想买小米"

每个人的购买理由都不同。"群众认可华为"是这些**个人决策的统计结果**，不是集体意志。

**第三层：价值判断的多样性**

华为产品的"价值"在不同人眼里完全不同：

| 人群    | 对华为手机的评价 | 原因          |
| ----- | -------- | ----------- |
| 技术发烧友 | 性价比低     | 同价位配置不如竞品   |
| 商务人士  | 值这个价     | 品牌溢价带来社交信号  |
| 爱国情怀者 | 非常值      | 支持国产技术的情感满足 |

**主观价值论的完整解释**：华为的高价能维持，是因为有足够多的人**主观上认为它值这个价**。

### "爱国营销"是个人选择的结果

你提到的"爱国营销"，用个人主义视角看：

**对企业**：选择"爱国"作为营销点，是一个**个人判断**——判断这个叙事能打动目标人群。

**对消费者**：接受"爱国"叙事并购买，也是一个**个人判断**——判断支持华为能带来情感满足。

**关键点**：双方都是自愿的。没有强迫，只有各自的主观价值判断。

奥派不说这是"合理"还是"不合理"，只说：**如果双方自愿交易，说明他们都认为自己在获益。**

### 方法论个人主义的边界

个人主义不等于"个人永远正确"。奥派承认：

1. **个人会犯错**
   - 买华为的人可能事后后悔
   - 做决策的华为员工可能判断错误
2. **信息可能不完整**
   - 消费者可能不知道同价位有更好的选择
   - 这是信息不对称，不是市场失败
3. **外部性存在**
   - 华为的营销可能影响第三方（如被攻击的小米）
   - 这涉及产权边界问题，需要另外讨论

### 为什么中央计划必然失败

方法论个人主义推导出奥派最著名的结论：**中央计划经济不可能成功。**

原因：

**1. 知识分散在无数个人手中**

- 北京的计划者不可能知道：云南某个小店的老板今天需要什么货
- 这个知识只在这个老板脑子里

**2. 个人偏好无法被统计**

- 你无法问 14 亿人各自想要什么
- 即使问了，人的偏好每天都在变

**3. 没有价格信号，无法协调**

- 市场经济中，价格告诉生产者"该生产什么"
- 计划经济没有这个价格信号，只能瞎猜

米塞斯在 1920 年的论文《社会主义国家的经济计算》中论证了这一点。20 世纪所有社会主义国家的实践，验证了这个预言。

### 商业启示

**对消费者**：

- 你的每一次购买都是**个人选择**
- 不要被"大家都在买"影响，问自己：这对我值多少钱？

**对生产者**：

- 不要试图满足"市场"，要满足**具体的个人**
- 找到那群**主观上认为你产品值这个价**的人

**对投资者**：

- "群众情绪"是无数个个人决策的统计结果
- 情绪可以持续很久，不要试图预测转折点

## 小结

1. 方法论个人主义：只有个人能行动，"集体"不能行动。
2. 所有经济现象都要还原到个人选择来解释。
3. 华为的高价能维持，是因为足够多的人主观上认为它值这个价。
4. "爱国营销"是企业和消费者各自个人选择的结果。
5. 方法论个人主义解释了为什么中央计划必然失败：知识分散，无法集中。
